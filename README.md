# GastoBot

Bot de Telegram para gestión de gastos personales y vencimientos automáticos.

## Funcionalidades

- Registro de gastos
- Categorías personalizadas
- Resumen mensual
- Gestión de vencimientos
- Recordatorios automáticos
- Dashboard con Grafana
- Persistencia en MySQL
- Docker Compose

## Tecnologías

- Python
- Flask
- SQLAlchemy
- MySQL
- Docker
- APScheduler
- Grafana
- Telegram Bot API
